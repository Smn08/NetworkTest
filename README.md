# Network Testing Tool

Инструмент для тестирования сетевых соединений с графическим интерфейсом.

## Системные требования

- Windows 10 или новее
- Docker Desktop для Windows
- VcXsrv (X Server для Windows)
- Права администратора

## Установка

1. Скачайте и распакуйте архив с программой в удобное место
2. Запустите `setup.bat` от имени администратора
   - Скрипт автоматически установит VcXsrv, если он не установлен
   - Настроит автозапуск X Server
   - Проверит наличие Docker Desktop
   - Настроит необходимые переменные окружения
   - Добавит правила в файрвол

3. После установки Docker Desktop:
   - Запустите Docker Desktop
   - Перезагрузите компьютер
   - Запустите `setup.bat` снова для завершения настройки

## Запуск программы

1. Убедитесь, что Docker Desktop запущен
2. Запустите `start.bat` от имени администратора
   - Скрипт проверит наличие всех необходимых компонентов
   - Запустит X Server, если он не запущен
   - Запустит контейнер с приложением
   - Откроет графический интерфейс

## Структура проекта

- `setup.bat` - скрипт первоначальной настройки
- `start.bat` - скрипт запуска приложения
- `docker-compose.yml` - конфигурация Docker
- `Dockerfile` - инструкции для сборки Docker-образа
- `gui.py` - основной файл приложения
- `requirements.txt` - зависимости Python

## Типы тестов

### 1. Флуд тесты
1. UDP Flood (1_1)
   - Отправка UDP пакетов с случайным размером данных
   - Случайный интервал между пакетами
   - Фрагментация пакетов
   - Параметры: `--port`, `--threads`, `--duration`

2. TCP SYN Flood (1_2)
   - Отправка TCP SYN пакетов
   - Случайный размер окна
   - Случайный TTL
   - Случайный тип сервиса
   - Параметры: `--port`, `--threads`, `--duration`

3. HTTP Flood (1_3)
   - Отправка HTTP запросов
   - Различные HTTP методы (GET, POST, HEAD и др.)
   - Случайные пути запросов
   - Реалистичные заголовки
   - Параметры: `--port`, `--threads`, `--duration`

4. ICMP Flood (1_4)
   - Отправка ICMP пакетов
   - Различные типы ICMP
   - Случайный размер данных
   - Случайный интервал
   - Параметры: `--threads`, `--duration`

### 2. Амплификация тесты

2. NTP Amplification (2_2)

## Устранение неполадок

### Проблемы с X Server
1. Убедитесь, что VcXsrv установлен
2. Проверьте, что X Server запущен (иконка в трее)
3. Перезапустите X Server вручную:
   ```
   "C:\Program Files\VcXsrv\vcxsrv.exe" -multiwindow -clipboard -wgl -ac
   ```

### Проблемы с Docker
1. Проверьте, что Docker Desktop запущен
2. Убедитесь, что служба Docker работает
3. Попробуйте перезапустить Docker Desktop

### Проблемы с запуском
1. Запускайте скрипты от имени администратора
2. Проверьте наличие всех необходимых файлов
3. Убедитесь, что все компоненты установлены корректно

## Обновление

1. Остановите все запущенные контейнеры
2. Удалите старую версию
3. Распакуйте новую версию
4. Запустите `setup.bat` для обновления конфигурации

## Поддержка

При возникновении проблем:
1. Проверьте раздел "Устранение неполадок"
2. Убедитесь, что все системные требования выполнены
3. Проверьте логи Docker и X Server
4. Обратитесь к разработчикам с описанием проблемы
